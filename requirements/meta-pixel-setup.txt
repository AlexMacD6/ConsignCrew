# Meta Pixel Setup with Conversions API Requirements

## Overview
Set up Meta Pixel with Conversions API for tracking Facebook Ads performance and conversions on TreasureHub.

## Key Features
- Meta Pixel base code integration
- Conversions API server-side tracking
- Event tracking for key user actions
- Custom conversion events for Facebook Ads optimization

## Environment Variables Required
Add the following to your `.env.local` file:
```
META_PIXEL_ID=your_meta_pixel_id_here
META_ACCESS_TOKEN=your_meta_access_token_here
META_API_VERSION=v18.0
```

## Events to Track
1. **PageView** - Track all page views
2. **Lead** - Track email signups and registrations
3. **ViewContent** - Track listing views
4. **AddToCart** - Track when users save/hide listings
5. **InitiateCheckout** - Track when users start the consignment process
6. **Purchase** - Track completed consignment agreements
7. **Contact** - Track contact form submissions

## Files to Create/Modify
- `app/lib/meta-pixel.ts` - Meta Pixel utility functions
- `app/api/meta-conversions/route.ts` - Conversions API endpoint
- Update `app/layout.tsx` - Add Meta Pixel base code
- Update key pages to trigger conversion events

## Dependencies
- No additional npm packages required (uses fetch API)

## Testing
- Test pixel firing in Facebook Events Manager
