# Meta Pixel Removal and Google Tag Manager Setup Requirements

## Overview
Remove all Meta Pixel (Facebook Pixel) code from the application and prepare for Google Tag Manager integration for better tag management and analytics.

## Meta Pixel Removal Requirements

### 1. Remove Meta Pixel Files
- Delete `app/lib/meta-pixel.ts` - Core Meta Pixel utility functions
- Delete `app/components/MetaPixelProvider.tsx` - React provider component
- Delete `app/api/meta-conversions/route.ts` - Meta Conversions API endpoint
- Delete `requirements/meta-pixel-setup.txt` - Meta Pixel setup requirements
- Delete `requirements/meta-pixel-error-fix.txt` - Meta Pixel error fix requirements
- Delete `pr-templates/meta-pixel-setup.md` - Meta Pixel setup PR template
- Delete `pr-templates/meta-pixel-error-fix.md` - Meta Pixel error fix PR template

### 2. Remove Meta Pixel References
- Remove MetaPixelProvider import and usage from `app/layout.tsx`
- Remove Meta Pixel tracking calls from `app/page.tsx`
- Remove Meta Pixel tracking calls from `app/contact/page.tsx`
- Remove Meta Pixel server-side events from `app/api/subscribe/route.ts`
- Remove Meta Pixel environment variables from `env.example`

### 3. Clean Up Environment Variables
- Remove `NEXT_PUBLIC_META_PIXEL_ID`
- Remove `META_ACCESS_TOKEN`
- Remove `META_API_VERSION`

## Google Tag Manager Preparation

### 1. Current GTM Setup
- Google Tag Manager is already configured in `app/layout.tsx`
